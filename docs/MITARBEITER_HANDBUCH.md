# ImmoRefi Portal - Mitarbeiter-Handbuch
**Einfache Anleitung für den täglichen Gebrauch**
*Stand: Januar 2026*

---

## Was ist das ImmoRefi Portal?

Das ImmoRefi Portal ist unsere zentrale Plattform für die Zusammenarbeit mit Kunden. Hier können Kunden:
- Termine buchen
- Bestellungen aufgeben
- Dokumente austauschen
- Mit uns chatten
- Verträge einsehen

Als Mitarbeiter/Admin haben Sie zusätzlich Zugriff auf:
- CRM (Kundenverwaltung)
- Alle Buchungen und Termine
- Rechnungen und Bestellungen
- Benutzerverwaltung

---

## Wie melde ich mich an?

1. Öffnen Sie **https://portal.immoportal.app**
2. Klicken Sie auf **"Login"**
3. Melden Sie sich mit Ihrer E-Mail an (oder Google-Konto)
4. Nach dem Login sehen Sie das Dashboard

**Tipp:** Speichern Sie die Seite als Lesezeichen!

## Die wichtigsten Aufgaben

### 1. Neue Kundenanfrage bearbeiten

Wenn ein neuer Lead reinkommt:

1. Gehen Sie zu **CRM → Leads**
2. Klicken Sie auf den neuen Lead
3. Prüfen Sie die Kontaktdaten
4. Rufen Sie an oder schreiben Sie eine E-Mail
5. Ändern Sie den Status:
   - **"Kontaktiert"** - wenn Sie Kontakt aufgenommen haben
   - **"Qualifiziert"** - wenn der Kunde Interesse hat
   - **"Verloren"** - wenn kein Interesse besteht

**Tipp:** Wenn der Lead zum Kunden wird, klicken Sie auf "Zu Kontakt konvertieren"

### 2. Deal in der Pipeline verwalten

Die Pipeline zeigt alle laufenden Projekte als Kanban-Board:

**Spalten:**
- **Neu** → Deal gerade angelegt
- **Qualifiziert** → Kunde hat Interesse bestätigt
- **Angebot** → Angebot wurde gesendet
- **Verhandlung** → In Abstimmung
- **Gewonnen** → Erfolgreich abgeschlossen 🎉
- **Verloren** → Nicht zustande gekommen

**Bedienung:** Ziehen Sie Deals per Drag & Drop in die richtige Spalte

### 3. Kundennachricht beantworten

1. Roter Badge bei "Nachrichten" zeigt neue Nachrichten
2. Klicken Sie auf **Nachrichten**
3. Wählen Sie den Kunden aus
4. Schreiben Sie Ihre Antwort
5. Klicken Sie auf **Senden**

**Wichtig:** Antworten Sie zeitnah - Kunden erwarten schnelle Reaktion!

### 4. Termin verwalten

1. Gehen Sie zu **Termine → Buchungen**
2. Sehen Sie alle gebuchten Termine
3. Nach dem Termin: Status auf **"Abgeschlossen"** ändern
4. Bei Nichterscheinen: Status auf **"No-Show"**

### 5. Dokument für Kunden bereitstellen

1. Gehen Sie zu **Verwaltung → Dokumente**
2. Klicken Sie auf **"Hochladen"**
3. Wählen Sie Datei und Kunde aus
4. Speichern - Kunde sieht das Dokument sofort

### 6. Bestellung bearbeiten

1. Gehen Sie zu **Verwaltung → Bestellungen**
2. Neue Bestellungen haben Status **"Neu"**
3. Ändern Sie den Status:
   - **"In Bearbeitung"** - Sie arbeiten daran
   - **"Warten auf Kunde"** - Sie brauchen Infos
   - **"Abgeschlossen"** - Fertig!

## Tipps für den Alltag

### ✅ Das sollten Sie tun:
- Täglich einloggen und Dashboard prüfen
- Nachrichten schnell beantworten (innerhalb von 24h)
- Leads zeitnah kontaktieren (am besten am selben Tag)
- Status immer aktuell halten

### ❌ Das sollten Sie vermeiden:
- Kundendaten per E-Mail versenden (nutzen Sie das Portal!)
- Leads liegen lassen ohne Kontaktaufnahme
- Bestellungen im Status "Neu" vergessen

## Club Deal System

Das Club Deal System ermöglicht Projektentwicklern, Kapital von professionellen Investoren einzuwerben (Private Placement, max. 18 Investoren, ab €100.000).

### Als Admin

Unter **`/admin/club-deals`** können Sie:
- Neue Projekte anlegen und bearbeiten
- Dokumente hochladen (Pitchdeck, Due Diligence, etc.)
- Projekte freischalten (`active` setzen)
- Zeichnungen bestätigen oder ablehnen

**Status-Workflow eines Projekts:**
`Entwurf → Prüfung → Aktiv → Vollständig finanziert → Abgeschlossen`

### Als Investor (Endkunde)

1. Unter `/investor` die Landing Page aufrufen
2. Selbstauskunft unter `/investor/onboarding` ausfüllen (dauert 2 Minuten)
3. Nach Admin-Freischaltung: Zugang zu allen aktiven Projekten unter `/investor/dashboard`
4. Zeichnung abgeben, Warteliste, Stornierung — alles unter `/investor/subscriptions`

### Technische Details

Vollständige Entwickler-Dokumentation: **`docs/CLUB_DEAL_SYSTEM.md`**
